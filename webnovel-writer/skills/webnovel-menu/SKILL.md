# Skill: /cnw - 网文助手交互式菜单

## 功能描述

提供中文交互式菜单，所有网文助手功能通过菜单选择执行，无需记忆复杂命令。

## 使用方式

```
/cnw
```

## 执行流程

1. 用户输入 `/cnw`
2. 显示主菜单
3. 用户输入数字选择（0-5）
4. 进入对应子菜单
5. 用户选择具体操作
6. 收集必要参数（章节号等）
7. 执行对应命令
8. 显示执行结果
9. 询问是否继续或返回

## 参数收集规则

| 操作 | 必要参数 | 收集方式 |
|------|---------|---------|
| 写章节 | 章号 | 请输入章号 |
| 生成大纲 | 卷号 | 请输入卷号 |
| 审查章节 | 范围 | 请输入章节范围（如 1-5 或 45） |
| 健康检查 | 章号/范围 | 选填，默认当前章节 |
| 读者反馈 | 章号、类型、内容 | 请依次输入 |
| 查询信息 | 关键词 | 请输入查询关键词 |

## 实现要求

- 使用 Python 交互式输入
- 中文字符对齐显示
- 错误输入时提示并重新输入
- 支持 Ctrl+C 中断返回主菜单
- 执行结果清晰展示

## 技术实现

该 Skill 通过 `scripts/menu_interactive.py` 实现菜单逻辑，调用 `scripts/data_modules/webnovel.py` 统一 CLI 执行实际操作。
